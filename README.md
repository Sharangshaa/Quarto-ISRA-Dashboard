## ISRA Dashboard: Manjapparai, Kelshi Uttambar & Kakinada

### Overview
This interactive dashboard presents key ecological and conservation insights from three ISRA sites â€” Manjapparai, Kelshi Uttambar, and Kakinada. It highlights species composition, conservation status, and site-specific info.

### Features
-  **Species Tables**: Filterable and searchable reactable tables for each site
-  **Red List Status**: Visual breakdown of IUCN categories
- **Site Comparison**: Cross-site summaries for Manjapparai, Kelshi Uttambar and Kommu Teku ISRAs
-  **PDF Report Link**:  Factsheets included in the summary section
-  **Custom Icons & Themes**: Styled with Bootstrap 5 and Quarto layout enhancements
